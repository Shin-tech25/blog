# 渡来人の編成化（部）

渡来人の編成化については、こちらの記事でまとめました。

[冊封体制と五世紀の東アジアの勢力図](/emperor-yuryaku)

渡来人とは、百済からヤマト政権にわたってきた王族や技術者たちです。鉄器や硬質の土器である須恵器の制作、機織りや金属工芸などの技術、漢字や儒教などの学術が伝えられます。

ヤマト政権はこうした渡来人技術者たちをグループ化し、鉄器づくりの「韓鍛冶部（からかぬちべ）」、須恵器づくりの「陶作部（すえつくりべ）」などに組織しました。

# 氏姓制度

ヤマト政権は、従えた豪族の統治に「氏姓制度」を用います。豪族たちの「氏」と言われた血縁関係を中心とするグループに「姓」といわれるその豪族の地位や政権内の立ち位置をしめす称号を与えて統治したことが特徴的です。

- 蘇我臣
- 物部連
- 大伴連
- 出雲臣
- 筑紫君
- 東漢直

これらの、左側（蘇我、物部、大伴、出雲、筑紫、東漢・・・）は血縁関係を表した氏です。

右側（臣、連、君、直、首）は政権内でのポジションを表した「姓」です。現代の姓は名字のことを表していましたが、当時は役割を表していました。

- 臣：中央の有力豪族
- 連：軍事・物の生産など特定の仕事を持つ豪族
- 君：地方の有力豪族
- 直：地方豪族
- 首：地方の小豪族
