# 概要

丹羽宇一郎「死ぬほど読書」を読んで。こちらの本の中で、「人がおすすめした本が自分にとって良い本とは限らない」という旨で書かれていましたが、あえて本の中で紹介されていた本をまとめました。そうは言っても、これらの本は古典名著であり、大変読み応えがある本であると思います。

こちらの記事にもおすすめの図書やなぜ社会人に勉強が必要であるかが記載されています。是非、参考にしてください。
