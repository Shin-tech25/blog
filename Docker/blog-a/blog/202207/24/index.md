# 概要

丹羽宇一郎「死ぬほど読書」を読んで。こちらの本の中で、「人がおすすめした本が自分にとって良い本とは限らない」という旨で書かれていましたが、あえて本の中で紹介されていた本をまとめました。これらの本は古典名著であり、そうは言っても、大体読み応えがある本であると思います。
