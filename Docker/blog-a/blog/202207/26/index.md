# 崇仏論争

物部氏は軍事を職掌としており、政権内での地位を示す姓は、「連（むらじ）」でした。一方、蘇我氏は近畿地方の有力豪族で姓は「臣」、財務担当で先進的な知識を持つ渡来人とも深く交流していました。

物部氏、蘇我氏の両者は百済から伝えられた仏教を崇拝するか否かで対立していました。

この対立のことを「崇仏論争」といいますが、純粋な崇仏・廃仏というよりも、政権内での主導権争いが背景にありました。

欽明天皇（大伴氏が擁立した天皇） → 敏達天皇（排仏派） → 用名天皇（崇仏派）と天皇の立場も入れ替わりました。用明天皇は蘇我稲目の孫、馬子からみれば甥であったことからも、崇仏・排仏はそのまま権力闘争だったことが伺えます。

# 蘇我氏 vs 物部氏

用明天皇が死去すると、587 年、後継をめぐって蘇我馬子と物部守屋による武力対決が起こります。結果として、軍事の専門家であった物部氏を蘇我氏が破り、物部氏は滅亡します。さらに馬子は自らが擁立した崇峻天皇を暗殺してしまいました。

自身が擁立した天皇を暗殺するなんてひどいですね。。。権力闘争とは今も昔も醜いのでした。。

# 推古天皇

崇峻天皇の死後、日本で最初の女帝である推古天皇が即位します。推古天皇はバランス感覚に優れた英明な君主であり、蘇我馬子と厩戸皇子との近郊を保ち、王権の維持を図ったと考えられています。

# まとめ

崇仏論争から見て取れるように、宗教が権力闘争の道具になったわけです。また、擁立した天皇を殺しているわけですからその執念は凄まじいです。日本ではもともと全てのものに神が宿るという神道の考え方がベースにあったわけですが、輸入してきた仏教の考え方はまだ浸透しておらず、道具にまでなってしまったと考えられます。

現代社会でも、日本では特定の宗教を信じていない無信教が多数派です。宗教を道具にするまではいかないですが、信じるということがどういう行為なのか腑に落ちない人は多いのでは。
